# <%= projectname %>

A Custom Agentic Mobile App (CAMA) project.

## Getting Started

This project was scaffolded with the CAMA native mobile template. It includes:

- An SFDX project structure with a `force-app` package directory
- A CAMA digital experience container with app metadata, build metadata, and a starter Home screen
- A scratch org definition for development

### Deploy to your org

```bash
sf project deploy start
```

### Preview in the Playground App

After deploying, use the Salesforce Mobile Publisher Playground app to preview your native mobile experience.
